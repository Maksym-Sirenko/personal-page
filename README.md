# Maksym Sirenko Portfolio

Personal portfolio website built with React, TypeScript, Vite, and SCSS modules.

The site presents selected projects, core frontend skills, and direct contact links in a clean, responsive format.

## Stack

- React
- TypeScript
- Vite
- React Router
- SCSS Modules

## Features

- English and Ukrainian UI
- Saved language preference in the browser
- Project gallery with modal details
- About and contact section with direct links
- Responsive layout for mobile and desktop

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```
